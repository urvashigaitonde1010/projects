To run backend:

-    Install Python
-    Install Django
-    Install Pillow library
-    Install cors-header library if you are running front-end locally (if hosting the website, you do not need it)

To run django project:

-    Open terminal in this(project) directory.
-    run "python manage.py runserver"

If you make any changes in models.py file in eSignApp directory for changes in database tabels then to export those changes in database, run :

-    run "python manage.py makemigrations"
-    run "python manage.py migrate"


To run Front-end:

-    Go to ./Front-end/Project/html and open "D_Sign_In.html"
